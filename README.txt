ETS2MP mods

This repository holds (hopefully) ETS2MP approved mods.
Because everybody has different likes I decided to split up the mod into multiple small mods which target a specific improvement area.

I will use the following structure in the Build folder:
\build
\build\ETS2\1.16
\build\ETS2\1.17
\build\ETS2\1.18

While you shouldn't experience any problems using a module 'compiled' for an earlier version, 
versions target to your current version might have some optimalisations.
